# Site Web Dynamique — Asaphs-Foods 🍕

## Contexte
Interface utilisateur d'un service de restauration en ligne permettant 
de consulter un menu, filtrer les plats et passer une commande.

## Fonctionnalités
- Affichage dynamique de 16 plats répartis en 6 catégories 
  (Pizza, Burger, Pâtes, Salade, Dessert, Boisson)
- Filtrage en temps réel : afficher uniquement les plats disponibles
- Gestion du stock : badge "Rupture", alerte stock faible (< 5)
- Sélection de plat avec mise en surbrillance visuelle
- Validation de commande avec décrémentation automatique du stock
- Interface responsive avec effets hover et transitions CSS

## Stack Technique
HTML5 | CSS3 | JavaScript Vanilla

## Structure du projet
├── index.html   # Structure de la page
├── index.css    # Styles et mise en page responsive
└── main.js      # Logique métier : rendu, filtrage, commande
